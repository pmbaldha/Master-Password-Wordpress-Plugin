Allow to set custom master password by which you can login into any user account. 

== Description ==
This plugin allows you to log in as any user, using any administrator\'s password or custom master password which is configurable. The user can still log in using their own password.

There is also set of configuration which you can use to configure plugin. 



== Installation ==
Download your WordPress Plugin to your desktop.
If downloaded as a zip archive, extract the Plugin folder to your desktop.
Read through the \"readme\" file thoroughly to ensure you follow the installation instructions.
With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
Go to Plugins screen and find the newly uploaded Plugin in the list.
Click Activate to activate it.
Go to Settings->Master Password to configure this plugin.

== Frequently Asked Questions ==
Q.Can I set admin password as master password?
Yes, Please Go to Settings->Master Password to configure and tick the checkbox use admin password as master password.

Q.Can I set custom master password other than admin password?
Yes.

Q.Can I disallow other admin user from using master password?
Yes.
